## Tic-Tac-Toe Web Game 🎮
- Webbaserad version av klassiska Tic-Tac-Toe
- Skapa spelare, spela mot varandra, spara resultat i scoreboard
- HTML, CSS, JavaScript, LocalStorage
- Dynamisk layout, modulär kodstruktur och responsiv design
- Länk till repo: [Tic-Tac-Toe Web Game](https://github.com/Noddan84/Inlamningsuppgift)

## Poker Hand Comparison 🃏
- Modulärt JavaScript-projekt för att jämföra pokerhänder och avgöra vinnare
- Stödjer alla pokerkombinationer: Royal Flush, Straight Flush, Four of a Kind, Full House, Flush, Straight, Three of a Kind, Two Pair, One Pair, High Card
- Vitest-tester för alla handtyper
- Modulär kodstruktur med Card, Hand, CompareHands och DeckOfCards
- Länk till repo: [Poker Hand Comparison](https://github.com/Noddan84/pokerhands-vite-test-start)

## Questionnaire 📝
- Ett tidigt skolprojekt i JavaScript för att skapa ett frågeformulär (questionnaire)
- Interaktivt frågeformulär med poäng för olika kategorier (Cat, Dog, Fish, Rabbit)
- Sparar resultat i JSON-filer och beräknar högsta/lägsta poäng
- Använder prompt-sync, Node.js och filhantering (fs)
- Reflektion: idag skulle jag förbättra kodstruktur med flera moduler, bättre objektstruktur och undvika duplicerad kod
- Länk till repo: [Questionnaire](https://github.com/Noddan84/Questionnaire.git)

## Java Test Examples ☕
- Enkla Java-projekt med **JUnit 5** och **Selenium WebDriver** för lärande och experiment
- Testar både **enkel matematisk logik** och **webbläsarinteraktion** på demo-sidor
- Maven-baserad struktur med `src/main/java` och `src/test/java`
- Demonstrerar grundläggande **unit tests**, **parameterized tests** och **Selenium browser automation**
- Projektet är mest för **träning och övning**, inte produktionskod
- Länk till repo: [Java Test Examples](https://github.com/Noddan84/java-junit-selenium-tests)
